Перед решением задач изучите теорию к данному уроку.

Примеры решения задач

Задача

Задача. Всем <h3> поставьте текст '!!!'.

Решение: 

$('h3').html('!!!');

Задача

Задача. Всем <h3> сделайте текст зеленого цвета.

Решение: 

$('h3').css('color', 'green');

Задача

Задача. Всем <h3> сделайте жирный текст зеленого цвета.

Решение: 

$('h3').css({color: 'green', fontWeight: 'bold'});

Задача

Задача. Получите и покрасьте в красный цвет все элементы с классом zzz.

Решение: 

$('.zzz').css('color', 'red');

Задача

Задача. Получите и покрасьте в красный цвет все заголовки.

Решение: 

$(':headers').css('color', 'red');

Задача

Задача. Получите и покрасьте в красный цвет все заголовки с классом zzz.

Решение: 

$(':headers.zzz').css('color', 'red');

Задача

Задача. Получите и покрасьте в красный цвет все заголовки НЕ с классом zzz.

Решение: 

$(':headers:not(.zzz)').css('color', 'red');

Задача . На each

Задача. Поменяйте всем <li> на странице их текст на их порядковый номер.

Решение: 

$('li').each(function (index, elem){
	$(elem).html(index);
});

Можно также вместо elem использовать this: 

$('li').each(function (index, elem) {
	$(this).html(index);
});

Задачи для решения

Задачи на html, text, css

Для решения задач данного блока вам понадобятся следующие методы: html, text, css.

Всем <h2> поставьте текст "!". Показать решение.

Решение: 

$('h2').html('!');

Всем <h3> сделайте красный цвет. Показать решение.

Решение: 

$('h3').css('color', 'red');

Всем <h2> сделайте голубой цвет и font-size: 30px. Показать решение.

Решение: 

$('h2').css({'color': 'blue', 'font-size': '30px'});

Задачи на простые селекторы, +, ~

Получите и покрасьте в красный цвет следующие элементы:

Все элементы с классом test. Показать решение.

Решение: 

$('.test').css('color', 'red');

Все <h2> с классом test. Показать решение.

Решение: 

$('h2.test').css('color', 'red');

Все абзацы которые следуют сразу за абзацами. Показать решение.

Решение: 

$('p + p').css('color', 'red');

Все <li>, которые следуют сразу за <li> с классом www. Показать решение.

Решение: 

$('li + li.www').css('color', 'red');

Все <i> внутри <h2>. Показать решение.

Решение: 

$('h2 i').css('color', 'red');

Все <i> непосредственно внутри <h2>. Показать решение.

Решение: 

$('h2 > i').css('color', 'red');

На селекторы jQuery

Для решения задач данного блока вам понадобятся следующие псевдоклассы jQuery: :has, :header, :empty, :contains, :not, :last, :first.
Получите и покрасьте в красный цвет следующие элементы:

Все заголовки. Показать решение.

Решение: 

$(':header').css('color', 'red');

Все <h2> НЕ с классом test. Показать решение.

Решение: 

$('h2:not(.test)').css('color', 'red');

Все заголовки НЕ с классом test. Показать решение.

Решение: 

$(':header:not(.test)').css('color', 'red');

Все заголовки, которые следуют сразу за заголовками. Показать решение.

Решение: 

$(':header + :header').css('color', 'red');

Все заголовки с классом test, которые следуют сразу за заголовками с классом test. Показать решение.

Решение: 

$(':header.test + :header.test').css('color', 'red');

Все пустые элементы. Показать решение.

Решение: 

$(':empty').css('color', 'red');

Первый абзац <p>. Показать решение.

Решение: 

$('p:first').css('color', 'red');

Последний абзац <p>. Показать решение.

Решение: 

$('p:last').css('color', 'red');

Все <li>, которые содержат текст 'пункт'. Показать решение.

Решение: 

$('li:contains("пункт")').css('color', 'red');

Все <h2>, внутри которых есть тег <i>. Показать решение.

Решение: 

$('h2:has(i)').css('color', 'red');

Все <h2>, которые непосредственно содержат <i>. Показать решение.

Решение: 

$('h2 > i').css('color', 'red');

Задачи на селекторы jQuery

Получите и покрасьте в красный цвет следующие элементы:
Для решения задач данного блока вам понадобятся следующие псевдоклассы jQuery: :even, :odd, :eq, :lt, :gt, :first-child, :last-child, :nth-child.

Третью <li>. Показать решение.

Решение: 

$('li:eq(3)').css('color', 'red');

Все <li> больше 10-той (отсчет с нуля в jQuery!). Показать решение.

Решение: 

$('li:gt(9)').css('color', 'red');

Все <li> меньше 10-той. Показать решение.

Решение: 

$('li:lt(9)').css('color', 'red');

Все <li> больше 10-той и саму 10-ю тоже. Показать решение.

Решение: 

$('li:gt(8)').css('color', 'red');

Получите первую <li> на странице. Показать решение.

Решение: 

$('li:first').css('color', 'red');

Получите последнюю <li> на странице. Показать решение.

Решение: 

$('li:last').css('color', 'red');

Получите первую <li> в каждой <ol>. Показать решение.

Решение: 

$('ol li:first').css('color', 'red');

Получите последнюю <li> в каждой <ol>. Показать решение.

Решение: 

$('ol li:last').css('color', 'red');

Получите 10-ю <li> в каждой <ol>. Показать решение.

Решение: 

$('ol li:eq(9)').css('color', 'red');

Четные <li>. Показать решение.

Решение: 

$('li:odd').css('color', 'red');

Нечетные <li>. Показать решение.

Решение: 

$('li:even').css('color', 'red');

Четные <li> в каждой <ol>. Показать решение.

Решение: 

$('ol li:odd').css('color', 'red');

Нечетные <li> в каждой<ol>. Показать решение.

Решение: 

$('ol li:even').css('color', 'red');

Задачи на parent()

Для решения задач данного блока вам понадобятся следующие методы: parent.

Сделайте родителя <li> с классом www (li с этим классом, а не родитель) красного цвета. Показать решение.

Решение: 

$('li.www').parent().css('color', 'red');

Задачи на each

Для решения задач данного блока вам понадобятся следующие методы: each.

Выведите алертом содержимое всех абзацев <p> (по очереди). Показать решение.

Решение: 

$('p').each(function(){
	alert($(this).html())
});

Выведите алертом содержимое всех заголовков (по очереди). Показать решение.

Решение: 

$(':header').each(function(){
	alert($(this).html())
});

Поменяйте содержимое всех <li> на их порядковый номер. Показать решение.

Решение: 

$('li').each(function(index){
	$(this).html(index)
});

Добавьте всем <li> в конец их порядковый номер. Показать решение.

Решение: 

$('li').each(function(index){
	$(this).append(index)
});
